* **자주 사용되는 Mathf함수**
  * Mathf.Deg2Rad
     * degree (60분법)을 통해 Radian(호도법)을 계산하는 코드
  * Mathf.Rad2Deg
     * 라디안값을 디그리 값으로 바꿔줍니다.
  * Mathf.Abs()
     * 절대값을 계산해주는 기능
  * Mathf.Atan
     * 아크 탄젠트 값을 계산합니다.
  * Mathf.Clamp(value,min,max)
    * value를 min과 max사이의 값으로 고정하는 기능
  * Mathf.Log10
    * 베이스가 10으로 지정되어있는 수의 로그를 반환해주는 기능
